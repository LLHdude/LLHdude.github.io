年关将至，过年走亲戚不可避（确信，很多时候不方便携带电脑主机，怎么样避免这段时间过于坐牢，保持折腾的态度，于是想到了可以用手机连接服务器。我尝试了大名鼎鼎的Android终端模拟软件Termux，简单配置并进行了一定的美化，使用体验很不错，在此记录。

Termux是什么

Termux 是一款Android平台下的终端模拟器，开源且不需要 root，因此绝大多数Android都可以轻松运行。Termux 使用 pkg(apt) 进行软件包的管理，完美支持 Python、 PHP、 Ruby、 Nodejs、 MySQL等（据说如此）。

它能做什么：模拟一个Linux环境，运行一些简单的Linux服务

  • 享受 Bash 和 Zsh
  • 使用 Vim 编辑文件
  • 通过 SSH 访问服务器
  • 使用 GCC 和 Clang 编译代码
  • 使用 Git 检查项目
  • 运行 MySQL，Redis 等服务器
  • …
