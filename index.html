VMD的基本原理变分模态分解（Variational Mode Decomposition, VMD）是一种自适应的信号处理技术，它能够将复杂信号分解成一组具有物理意义的模态函数（Intrinsic Mode Functions, IMFs）。与经典的经验模态分解（Empirical Mode Decomposition, EMD）相比，VMD具有更高的精度和稳定性，适用于各种信号分析任务。 VMD通过优化问题来寻找一组模态函数，这些模态函数的组合能够最好地近似原始信号，同时满足一定的约束条件。这些约束条件包括带宽限制、中心频率的分布以及噪声容忍度等。 2. #PPG信号相关（二）——信号预处理 1. 预处理流程 2. 下采样（可选）下采样是减少信号采样率的过程。在数字信号处理中，原始信号通常以较高的采样率进行采样，以便捕捉到所有的频率分量。然而，在某些情况下，较高的采样率可能会导致数据量过大，处理和存储成本增加。这时，下采样就显得非常有用。 由于PPG信号往往用于可穿戴设备上，因此有必要减少数据量，以减小存储和处理的成本。 下采样的基本步骤包括： 低通滤波：在进行下采样之前，首先需要对信号进行低通滤波处理。这是因为奈奎斯特定理指出，为了避免混叠，采样频率必须至少是信号最高频率的两倍。下采样会降低采样率，因此需要通过低通滤波器去除高于尼奎斯特频率的频率分量，以防止混叠现象。 采样点选择：滤波后，从滤波输出中每隔一定间隔选择一个采样点。这个间隔称为下采样因子。例如，如果原始信号的采样率为4倍于信号最高频率，而我们希望将其降低到2倍，那么我们可以选择每隔一个采样点进行下采样，下采样因子为2。 下采样的主要优点是减少了数据量，降低了存储和处理的成本。然而，它也可能导致信号中一些重要信息的丢失，特别是当信号包含高频分量时。 3. 信号去噪该步 ...
